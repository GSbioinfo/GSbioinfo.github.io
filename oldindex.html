About Me

scDNA-seq analysis of CRISPR edited cells

his repo contains a software tool and Dockerfiles for analysis of single-cell CRISPR editing data generated on the Tapestri platform. Also provided are R scripts for analysis visualization, and python scripts for testing Machine learning models.

Vector Integration Site Analysis (VISA)

This tool processes and analyzes vector integration site data. Implemented in C++/Python/SQL for processing raw sequence data from Illumina MiSeq and iSeq platforms. QVISA.

PetalPlots

This Perl script creates petal plots to visualize three-dimensional data in 2D, simplifying data interpretation. PetalPlots for detailed guidelines.

scRNA-seq of human HSPC

This repository contains R scripts and Dockerfiles for single-cell analysis of human bone marrow CD34 cells. HSPC data analysis for details.
