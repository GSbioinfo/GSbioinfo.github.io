Vector integration site analysis (VISA)

This software tool proecess and analyzes vector integration site data. It is implemented in C++/Python/SQLite for processing raw sequence data generated on Illumina MiSeq and iSeq platforms. QVISA.

Petal Plots

This perl script create petal plots to visualize three dimentional data in 2D plane that simplifies data interpretation. PetalPlots for detailed guidelines.

scRNA-seq of human HSPC

This repo contains R scripts and Dockerfile for analysis of human bone marrow CD34 cells. HSPC data analysis for details.

scDNA-seq analysis of CRISPR edited cells

This repo contains software tool developed and Dockerfiles for analysis of single cell CRISPR editing data generated on Tapestri platform. More details comming soon...
